**Installations**

To get started, you must be run **project.sh** file. 


After installation, go to the project folders where test you want to run.

If you want to run TEST-1 ;

**jest api1_spec.js**

If you want to run TEST-2 ;

**jest api2_spec.js**

If you want to run TEST-3 ;

**jest api3_spec.js**
